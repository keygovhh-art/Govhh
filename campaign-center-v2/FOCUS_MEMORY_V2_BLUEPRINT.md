# GAVHAH Focus Memory V2

## The product insight

A focus system is incomplete if it only asks what must happen today.

It must also remember:

- What was planned yesterday?
- Which of the main moves were completed?
- What was carried forward?
- Which blocker keeps returning?
- What worked during the last three days?
- What changed during the last seven or thirty days?
- How has the campaign moved since launch?

The result is not merely a task list. It is organizational memory.

## Shared foundation

Every paid campaign plan receives:

- Today’s shared outcome
- Main focus moves
- Owner, status and due time
- Main blocker
- End-of-day win
- Lesson learned
- Daily historical snapshot
- Previous-day and campaign-lifetime view

Personal Askan remains unchanged and private.

## Chesed Quick

Fixed, simple structure:

- One focus group
- Three main campaign moves
- Yesterday view
- Last three days
- Last seven days
- Campaign lifetime summary
- End-of-day review

The value is simplicity and shared direction.

## Askan Pro

Boosted control:

- Choose three or five main moves
- Choose one or two focus groups
- Seven-day campaign sprint
- Thirty-day history
- Carryover memory
- Repeated-blocker detection
- Success-pattern insights
- Weighted priorities

The value is choosing the next best move rather than merely tracking activity.

## Gabbai Pro

Operations memory:

- Choose three, five or seven main moves
- Up to three focus groups
- Operations-lane history
- Waiting-on records
- Handoff history
- Team-load view
- Repeated operational bottlenecks
- Lifetime campaign history

The value is seeing what repeatedly becomes stuck between people and teams.

## Organization

Boss and leadership controls:

- Choose three, four, five or ten main moves
- Preset: three groups of three
- Up to four focus groups
- Rename every group
- Portfolio history
- Cross-campaign trends
- Leadership decision queue
- Resource-allocation memory
- Campaign lifetime from launch until completion

The value is preserving why leadership made decisions, not only what was decided.

## Custom

Custom mission memory:

- One to twenty priorities
- One to five focus groups
- Custom group names
- Custom scoring formulas
- Custom review cadence
- API and hotline events
- Managed review
- Custom organizational signals

## Historical dashboard

Available time views depend on plan:

- Yesterday
- Last three days
- Last seven days
- Last thirty days
- Campaign lifetime

Every historical day stores:

- Shared outcome
- Planned main moves
- Completed moves
- Carryovers
- Main blocker
- Win
- Lesson
- Success rate

## Privacy boundary

The shared historical dashboard stores explicit campaign facts.

It must never absorb:

- Personal brain dumps
- Private personal focus
- Private notes
- Unshared personal planning

Personal memory and campaign memory are two separate systems.
